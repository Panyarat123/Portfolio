Thai NOK — Defect Tracking Dashboard

Role: BI Developer & Data Analyst

จำลองระบบ Paperless Defect Tracking สำหรับโรงงานผลิตชิ้นส่วนยาง โดยออกแบบ Data Flow ตั้งแต่การเก็บข้อมูลจากไลน์ผลิต ไปจนถึงการแสดงผลใน Power BI Dashboard สำหรับผู้จัดการโรงงาน

🔍 Business Problem:

พนักงานหน้าไลน์ผลิตยังบันทึกของเสียลงกระดาษ ทำให้ข้อมูลล่าช้า ผิดพลาด และผู้บริหารไม่สามารถเห็นภาพรวมได้แบบ Real-time

📊 Dashboard Features:

  • Card: ยอดรวมของเสียทั้งหมด
  • Donut Chart: สัดส่วนประเภทของเสีย (Air Bubble, Tear, Dimension Error, Scratch)
  • Bar Chart: เปรียบเทียบของเสียแต่ละเครื่องจักร (Molding-01, 02, 03)
  • Column Chart: เปรียบเทียบกะเช้า vs กะดึก
  • Slicer: Interactive Filter กดเลือกดูแต่ละเครื่องจักรได้

💡 Key Insights:

  • Molding-01 มีของเสียมากที่สุด → ควรตรวจสอบเป็นอันดับแรก
  • Dimension Error (26.37%) คือสาเหตุที่เกิดบ่อยที่สุด → อาจต้อง Calibrate เครื่องจักร
  • กะดึก มีของเสียมากกว่ากะเช้าเล็กน้อย → ควรติดตามอย่างใกล้ชิด
